# Catarak Sense

Aplikasi deteksi katarak berbasis web.

## Struktur Folder
- `public/` : File publik & service worker
- `src/assets/` : Asset gambar/dll
- `src/scripts/` : Script JavaScript utama
- `icons/` : Ikon aplikasi
- `index.html` : Halaman utama
- `style.css` : CSS utama

## Cara Menjalankan
1. Install dependency:  
   ```
   npm install
   ```
2. Jalankan project:  
   ```
   npm run dev
   ```
